#  Interactive Book Showcase

##  Overview
A React web application that fetches book data from the Google Books API and provides:
- Book list with search functionality
- Detailed book view
- Favorite books functionality (saved in localStorage)
- Navigation with React Router

##  Features
- Fetch books using Google Books API
- Search books by title/author
- View book details (title, author, description, cover)
- Add/remove favorites (persisted in localStorage)
- Favorites page to display saved books
- Responsive UI with reusable components
